# Vendored report assets

These files are inlined into the self-contained steering-evaluation HTML
report (`astrameter.simulator.eval_report`) so it renders offline with no
network or CDN dependency.

- `uPlot.iife.min.js`, `uPlot.min.css` — [uPlot](https://github.com/leeoniya/uPlot)
  v1.6.31, MIT License © Leon Sorokin. A small, fast canvas charting library
  for time-series. Update by re-downloading the pinned version from
  `https://cdn.jsdelivr.net/npm/uplot@<version>/dist/`.
